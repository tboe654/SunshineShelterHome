---# Sunshine Shelter Home

**Sunshine Shelter Home** is a responsive website created to showcase and support an animal rescue shelter. The website allows users to view services, browse the gallery, submit enquiries, and contact the shelter. Part 3 of the project adds interactive features, dynamic content, and client-side form validation.

## **Project Description**

Sunshine Shelter Home is designed to provide an accessible and interactive experience for users interested in:

- Adopting pets
- Volunteering or donating
- Viewing gallery images of rescued animals
- Contacting the shelter for enquiries

Part 3 of the project includes:

- Dynamic services page with search functionality
- Interactive gallery with lightbox
- Interactive map showing shelter location
- Enquiry and contact forms with JavaScript validation
- SEO optimization and responsive design

## **Features**

### **Navigation**
- Responsive navigation bar with links to all main pages

### **Hero Section**
- Main landing area on the home page with welcome content

### **Services**
- Dynamic content loaded from JavaScript
- Search bar for filtering services in real-time

### **Gallery**
- Interactive lightbox modal for images
- Hover effects and responsive layout

### **Interactive Map**
- Map displaying the shelter’s location
- Clickable marker with popup info

### **Forms**
- **Enquiry Form:** Validates user input and provides feedback
- **Contact Form:** Validates user input and simulates email feedback
- **Client-side validation** ensures correct format and required fields

### **SEO**
- Proper `<title>` tags, meta keywords, and description
- Descriptive `alt` text for all images
- Clean URL structure
